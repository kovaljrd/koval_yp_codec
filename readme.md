# Приложение для шифрования

для реализации был взят windows forms (.net framework) на C#

Функционал:
Шифрование/Дешифрование через шифры:Цезаря/rot-n с настройкой сдвига; base32/64; ascii; бинарный код; код-морзе; A1Z26

Подпись сообщения: подпись текста через SHA-256 с возможностью проверки подписи

История операций шифрования: просто логирование предыдущих действий, каждой операции выдаётся свой айди в виде "частот", ибо тематика приложения изначально должна была быть в стиле радио/кодека

Журнал всех действий: не требуется в описании.

Быстрое шифрование: изменение последнего текста из буфера обмена через горячую клавишу ctrl+shift+e

Tray-система: закрывание окна убирает процесс в трей и даёт открыть его обратно.

Использованные библиотеки:
Newtonsoft.Json - для работы с жсоном, наверное

Fody и Costura.Fody - для того чтоб вложить используемые библиотеки внутрь собранного .exe файла

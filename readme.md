# Змеиный кодек - приложение для шифрования данных

## Описание
Многофункциональное приложение для шифрования текста с тематическим интерфейсом кодек-радио.

## Системные требования
- ОС: Windows 7 и выше
- .NET Framework 4.8
- Разрешение экрана: от 854x480

## Установка
1. Скачайте архив с программой
2. Распакуйте в любую папку
3. Запустите файл `koval_yp_codec.exe`

## Функциональные возможности
- Шифрование/дешифрование 8 различными алгоритмами
- Цифровая подпись сообщений
- История операций с уникальными "частотами"
- Журнал всех действий
- Быстрое шифрование горячей клавишей Ctrl+Shift+E
- Работа в системном трее

## Архитектура приложения
Приложение построено по модульному принципу:
- `MainForm` - главная форма и навигация
- `Ciphers` - модуль шифров
- `Signature` - цифровая подпись
- `RadioHistory` - история операций
- `Logger` - журналирование
- `HotkeyManager` - глобальные горячие клавиши

## Используемые библиотеки
- Newtonsoft.Json - сериализация данных
- Fody/Costura.Fody - внедрение зависимостей в .exe

## Инструкция пользователя
1. **Главное меню** - выбор раздела
2. **Шифрование** - ввод текста, выбор алгоритма, получение результата
3. **Подпись** - создание и проверка цифровых подписей
4. **История** - просмотр предыдущих операций
5. **Журнал** - лог всех действий

## Руководство программиста
### Сборка проекта
1. Открыть решение в Visual Studio 2019+
2. Восстановить NuGet-пакеты
3. Выбрать конфигурацию Release
4. Собрать проект

### Добавление нового шифра
1. Добавить метод в класс `Ciphers`
2. Добавить название в `ComboBox` на вкладке шифрования
3. Добавить вызов метода в `BtnEncrypt_Click` и `BtnDecrypt_Click`

### Модификация интерфейса
Все элементы интерфейса создаются программно в методах `Create*Panel`. 
Для изменения расположения достаточно скорректировать координаты.

## Лицензия
Свободное использование в учебных целях.
